# 0.0.1

* Базовая инициализация проекта
* Добавлен Swagger OpenAPI
* Добавлены Dockerfile и Docker Compose
* Добавлены настройки подключения к Qdrant
* Добавлена конечная точка для моделей, совместимых с OpenAI
* Добавлены контракты API для управления документами
* Добавлены парсеры TXT и Markdown

* Добавлена потоковая передача ответов (stream=true, SSE)
* Реализована асинхронная обработка документов (CompletableFuture)
* Добавлен парсер PDF (Apache PDFBox)
* Добавлен парсер DOCX (Apache POI)
* Реализовано кэширование эмбеддингов
* Добавлен веб-поиск как fallback через SearXNG
* Реализован гибридный подход к работе с Qdrant (Spring AI + Qdrant Client)
* Добавлена обработка ошибок через GlobalExceptionHandler
* Улучшен PromptBuilder с жесткими правилами ответа по контексту
